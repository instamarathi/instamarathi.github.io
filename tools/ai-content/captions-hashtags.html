Captions आणि Hashtags — AI ने बनवा

चांगला visual असून caption boring असेल तर post पुढे जात नाही. Caption formula शिका, LLM ने execute करा.

Caption का महत्त्वाचं आहे?

Instagram वर visual पहिल्यांदा दिसतो — पण caption वाचल्यावर action होतो. "Like करायचं का? Save करायचं का? Comment करायचं का?" — हे caption ठरवतो. Algorithm ला comments, saves, shares आवडतात. Caption ने हे घडवायला हवं. त्यामुळे caption हा नंतरचा विचार नाही — तो content चा मुख्य भागच आहे.

Caption Formula

प्रत्येक caption च्या 4 parts असतात. हे formula आधी समजा, मग LLM ला सांगा:

  1. Hook (ओळ 1): पहिली ओळ — हीच "More" दाबण्यापूर्वी दिसते. "More" क्लिक केल्याशिवाय बाकी caption दिसत नाही. त्यामुळे hook line जोरदार असली पाहिजे — प्रश्न, एक जोरदार वाक्य, किंवा आश्चर्यकारक गोष्ट. जास्तीत जास्त 10 शब्द.
  2. Value (ओळ 2-6): वाचणाऱ्याला काय मिळतं? Post मध्ये काय शिकले, काय जाणवलं — त्याला शब्द द्या. Carousel साठी: "Swipe केलंत का? कोणती point जास्त लागू वाटली?" Reel साठी: मुख्य शिकवण लिहा.
  3. CTA (शेवटच्या आधीची ओळ): एकच स्पष्ट कृती. Save करा, comment करा, DM करा, tag करा — एक निवडा. "Like, share, save, comment, follow" — सगळं एकाच वेळी नको.
  4. Hashtags (शेवटचा भाग): 8-15 hashtags, वेगवेगळ्या size मध्ये. Line breaks ने गोंधळ कमी करा.

Caption Prompt

Dental carousel उदाहरण वापरून caption generate करा:

Caption PromptWrite an Instagram caption for a carousel post about root canal myths — "Root canal भयंकर नाही — 5 गोष्टी जे तुम्हाला माहीत असाव्यात." Client: Dr. Priya Smile Clinic, Pune. Dental clinic. Audience: Adults 25-45 who avoid dentists because of fear. Tone: Warm, reassuring, friendly — like a knowledgeable friend, not a marketing pitch. Caption structure: 1. Hook line (first line only — make someone stop before they scroll past) 2. 3-4 lines of value — expand on why this carousel matters, what emotion it addresses 3. One clear CTA — ask them to save this post OR comment their biggest dental fear 4. End with 2 line breaks, then hashtags (I'll add separately) Keep it under 150 words (excluding hashtags). Conversational. Can mix Marathi/Hindi naturally — don't force it.

Output आल्यावर वाचा: hook line जोरदार आहे का? CTA स्पष्ट आहे का? लांबी ठीक आहे का? जे नाही ते बदला — त्याच conversation मध्ये: "Hook line जास्त भावनिक करा" किंवा "CTA बदला — save पेक्षा comment करायला लावणारं असं."

Hashtag Prompt — Size मिक्स पद्धत

Hashtags म्हणजे कोणतेही शब्द टाकणं नाही. पद्धत म्हणजे: मोठे, मध्यम, छोटे, branded — मिक्स करायचं. मोठे hashtags (#fitness — 500M posts) मध्ये तुमचा post 2 मिनिटांत दिसेनासा होतो. छोटे hashtags (#PuneDentist — 50K posts) मध्ये तुमचा post तासनतास दिसतो. मिक्स केल्याने reach चांगला मिळतो.

Hashtag PromptGenerate 8 Instagram hashtags for a dental clinic post about root canal myths. Client: Pune-based dental clinic. Instagram audience: urban Indians 25-45. Mix: - 1 large hashtag (1M+ posts) — broad dental/health - 3 medium hashtags (100K-500K posts) — dental care, oral health India - 3 small hashtags (10K-100K posts) — location-specific or niche-specific (Pune, Maharashtra, specific treatment) - 1 branded hashtag — create one for Dr. Priya Smile Clinic For each hashtag, tell me: - The hashtag - Approximate post count (large/medium/small) - Why it fits this post Format as a numbered list.

Output येईल. तपासा: branded hashtag (#DrPriyaSmileClinic किंवा तसंच काही) client ला दाखवा आणि त्यांना नेहमी वापरायला सांगा — हे हळूहळू search मध्ये दिसायला लागतं. Location hashtags (#PuneDentist, #PuneHealth) जवळच्या audience साठी महत्त्वाचे.

Hashtag Tips — Cheat Sheet

  • बदलत राहा: एकाच set चे hashtags 10+ posts वर वापरणं = shadowban चा धोका. दर आठवड्याला 20-30% बदला.
  • Copy-paste नको: Client A चे hashtags Client B साठी वापरू नका. Niche, city, audience — सगळं वेगळं असतं.
  • Caption मधून वेगळे करा: 2-3 line breaks नंतर hashtags टाका. गोंधळ कमी होतो, caption वाचता येतो.
  • बंद केलेले hashtags टाळा: Instagram काही hashtags बंद करतो. Prompt मध्ये टाका: "Avoid banned or flagged hashtags."
  • Marathi/Hindi hashtags: स्थानिक audience साठी #पुणे #मराठी_content असे hashtags टाका — LLM ला नक्की मागा.
काम करण्याचा Tip: Caption आणि hashtags एकाच prompt मध्ये मागणं शक्य आहे — पण वेगळे ठेवलेलं better. Caption तपासल्यावर hashtag generate करा. यामुळे hashtags caption च्या final topic शी जुळतात, आणि बदलणं सोपं होतं.

Exercise — तुमच्या Carousel साठी Caption + Hashtags

मागच्या page वर (Carousel Script) तुम्ही dental carousel script बनवलं. त्याच topic साठी:

  1. Caption prompt वापरा — topic adjust करा (root canal myths carousel). Caption generate करा.
  2. Hook line वाचा — जर generic वाटली तर LLM ला सांगा: "Hook stronger करा — fear address करणारी असावी." Revised version मागवा.
  3. Hashtag prompt वापरा — same dental niche, Pune location. 8 hashtags generate करा, size mix सांगा.
  4. Final caption + hashtags एकत्र करा. एकदा पूर्ण वाचा — tone consistent आहे का, CTA clear आहे का?
