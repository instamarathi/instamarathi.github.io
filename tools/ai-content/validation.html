Content Validation — LLM ला Quality Checker बनवा

Script तयार आहे — पण publish करण्यापूर्वी 5 checks. LLM ला reviewer म्हणून वापरा. Client ला deliver करण्यापूर्वी हे करा.

Validation का करायचं?

Content बनवताना आपण त्यात involved असतो — त्यामुळे obvious problems दिसत नाहीत. Hook weak आहे पण आपल्याला "ठीक वाटतो." Slide 3 confusing आहे पण आपण "सगळं माहीत असल्याने" confusing वाटत नाही. LLM ला fresh perspective असतो — तुमचा script paste करा, specific questions विचारा, honest feedback मिळतो. हे 5 validation prompts एक checklist म्हणून वापरा — एकही skip करू नका.

Validation 1 — Hook Strength Check

Scroll-stop hook नाही = post fail. हा check सगळ्यात आधी करा:

Validation Prompt 1 — Hook StrengthI'm going to share the hook line (first line) of my Instagram carousel. Rate it on 3 dimensions, each out of 5: 1. Scroll-stop power: Would someone stop scrolling to read more? (1 = no reason to stop, 5 = irresistible) 2. Curiosity gap: Does it make them want to know what comes next? (1 = tells everything upfront, 5 = creates strong desire to swipe) 3. Specificity: Is it specific enough to feel real, not generic? (1 = could apply to anyone, 5 = feels written for exactly this audience) For each rating, give one sentence of reasoning. Then suggest one improved version of the hook that would score higher. My hook: [PASTE YOUR HOOK LINE HERE] Context: This is for a dental clinic carousel about root canal myths. Audience: urban Indians 25-45 who avoid dentists out of fear.

Score 4+ on सगळ्या 3 dimensions तर proceed करा. Score 3 किंवा कमी असेल तर LLM ने दिलेला improved version वापरा, किंवा alternatives मागवा.

Validation 2 — Brand Voice Compliance

Client ने brand voice describe केला असेल — "warm but professional," "not salesy," "educational." हे actually content मध्ये reflect होतंय का? LLM ला check करायला सांगा:

Validation Prompt 2 — Brand Voice CheckI have defined brand voice rules for this client. Check my carousel script against each rule. Brand voice rules: - Warm and reassuring, not clinical or scary - Educational tone — inform, don't sell - Conversational — like a knowledgeable friend, not a doctor lecturing - Never use aggressive sales language ("Book now!" "Limited offer!") - Can mix Marathi/Hindi naturally, but don't force it My carousel script: [PASTE FULL CAROUSEL SCRIPT HERE] For each brand voice rule: - Does the script follow it? (Yes / Partially / No) - Which specific line or slide violates or weakens it? - Suggest a rewrite for any violations. Be honest. Flag even small mismatches.

Output बघा: कोणत्या lines flagged आहेत? सगळ्या "Yes" असतील असं नाही — "Partially" असलेल्या lines rewrite करा. हे quality standard maintain करतं.

Validation 3 — Audience Relatability

Script technically correct आहे, brand voice आहे — पण audience ला relate होईल का? Outsider perspective महत्त्वाचं आहे:

Validation Prompt 3 — Audience RelatabilityImagine you are an urban Indian adult, 30 years old, living in Pune. You have been avoiding the dentist for 2 years because you're scared it will hurt and you don't know what to expect. You see this carousel on Instagram. Read this script as that person: [PASTE CAROUSEL SCRIPT HERE] Answer honestly: 1. Does the hook make you stop? What emotion do you feel in the first 3 seconds? 2. Does the content feel like it understands your situation — or does it feel generic? 3. Is there any slide where you'd lose interest and not swipe further? Which one and why? 4. After reading the full carousel, what action would you take — save, comment, DM, or nothing? Why? 5. One thing the creator could do to make this more relatable for someone like you.

हा prompt LLM ला persona मध्ये टाकतो — output जास्त honest येतो. "Slide 3 ला lose interest" असं सांगेल तर त्या slide ला improve करा.

Validation 4 — Comparison Validation

जर तुम्ही एकाच carousel साठी 3 different versions बनवल्या असतील (different hooks, different structures), तर comparison करा:

Validation Prompt 4 — Version ComparisonI have 3 versions of the same carousel hook. Rank them from best to worst for Instagram performance. For each, tell me: - Rank (1 = best) - Why it would perform better or worse - What specific word or phrase is making the difference Audience: urban Indians 25-45, fear dentists, see this on Instagram. Version A: [PASTE VERSION A HOOK] Version B: [PASTE VERSION B HOOK] Version C: [PASTE VERSION C HOOK] Be specific. Don't just say "Version A is more engaging" — say exactly why.

Use Case: तुम्ही Step 2 (Carousel Script page) मध्ये 5 hook alternatives generate केल्या होत्या. त्यातल्या top 3 इथे compare करा. LLM चा ranking तुमच्या gut feeling शी match होतो का? मग proceed करा.

Validation 5 — Factual Accuracy

Medical, legal, financial niches साठी हे सगळ्यात critical check आहे. चुकीची माहिती = client चं reputation damage = तुमचा contract गेला.

Validation Prompt 5 — Factual AccuracyReview this carousel script for factual accuracy. This is for a dental clinic's Instagram — content will be seen by the general public. [PASTE CAROUSEL SCRIPT HERE] Check: 1. Are all medical/dental claims accurate? Flag anything that is exaggerated, misleading, or not evidence-based. 2. Are there any statements that could cause harm if a patient follows them incorrectly? 3. Are there claims that need a disclaimer (e.g., "consult your dentist before...")? 4. Rate overall factual reliability: High / Medium / Needs Expert Review Note: I will also have the client's doctor review this before publishing. But flag anything that seems questionable.
Important: Factual validation खूप important आहे. Dental, medical, skincare, financial — या niches मध्ये चुकीची माहिती publish झाली तर client चं reputation damage होतं. LLM चा check हा first filter आहे — final approval नेहमी client / expert कडून घ्या. "Doctor ने approve केलं का?" हे delivery आधी विचारा.

5 Checks — Quick Reference

# Check What it catches Pass criteria
1 Hook Strength Weak, generic, no curiosity 4+ on all 3 dimensions
2 Brand Voice Off-tone lines, salesy language All rules: Yes or Partially fixed
3 Audience Relatability Generic feeling, wrong language, irrelevant Persona would save or comment
4 Version Comparison Chose weaker version intuitively Best version confirmed
5 Factual Accuracy Wrong claims, missing disclaimers High reliability + client review

Exercise — Script ला 5 Checks मधून Pass करा

Carousel Script page वर तुम्ही dental carousel script बनवला. तोच script घ्या आणि या 5 validation prompts मधून pass करा — एक-एक, same conversation मध्ये.

नोट करा: कोणत्या check मध्ये जास्त issues आले? Script publish ready आहे असं कधी वाटलं? हा exercise तुमचा "quality bar" set करतो — पुढे प्रत्येक script साठी हाच bar maintain करा.
