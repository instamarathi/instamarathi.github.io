Reel Script Writing — 30, 60, 90 Seconds

Reels carousel पेक्षा वेगळे असतात — pace, structure, delivery सगळं different. Right prompt दिला तर LLM format-specific scripts देतो.

Reel Structure — हे लक्षात ठेवा

कोणत्याही length च्या reel मध्ये same core structure असतो — duration बदलतो, framework नाही. हे framework LLM ला prompt मध्ये सांगायचं:

  • Hook (0-3 seconds): पहिल्या 3 seconds मध्ये viewer थांबला नाही तर reel fail. Hook visual असतो + text/audio. Question, bold statement, किंवा surprising visual — तीनपैकी एक.
  • Problem (3-15 seconds): Viewer ला relate होणारी problem state करा. "तुम्ही हे करत असाल तर..." format. Viewer ला वाटलं पाहिजे "हे माझ्याबद्दल बोलतोय."
  • Solution (15 sec to near end): Answer द्या — specific, actionable. 30-sec reels मध्ये एक solution, 60-90 sec मध्ये multiple points.
  • CTA (last 3-5 seconds): काय करायचं — follow, save, comment, DM. एकच CTA, clear.

30-Second Reel Script Prompt

Short reels साठी एकच strong point, sharp hook, direct CTA. Gym trainer niche example:

30-Sec Reel PromptWrite a 30-second Instagram Reel script for a gym trainer in Pune, India. Topic: "3 mistakes beginners make in their first week at the gym" Audience: Men and women aged 18-30 who just joined a gym but feel lost, intimidated, or aren't seeing results after 2-3 weeks. Format: Text-on-screen (not voiceover). Keep each text segment short — max 5-6 words per frame. Fast-paced cuts. Structure: - 0-3 sec: Hook (on-screen text + action shot) - 3-20 sec: 3 mistakes — one per quick cut (name the mistake, one line why it's wrong) - 20-28 sec: Quick fix for each - 28-30 sec: CTA Tone: Direct, energetic, a little tough-love — like a good trainer talks to a beginner. Not preachy. Label each section with timestamp and what appears on screen.

60-Second Reel Script Prompt

60-sec reels ला थोडा breathing room असतो. Same topic, more depth — explanation, brief demo, more reassuring tone:

60-Sec Reel PromptWrite a 60-second Instagram Reel script for a gym trainer in Pune, India. Topic: "3 mistakes beginners make in their first week at the gym — and exactly how to fix them" Audience: Same as before — beginners, feeling lost, maybe gave up once before. Format: Voiceover + text on screen. Script should be written as spoken words (natural, conversational — as if trainer is talking to a friend). Also note what appears on screen at each point. Structure: - 0-3 sec: Hook — ask a question the viewer is thinking - 3-10 sec: Brief setup — "Most beginners do these 3 things and then quit..." - 10-40 sec: Mistake 1, 2, 3 — each with: what the mistake is (5 sec), why it happens (5 sec), quick fix (3 sec) - 40-55 sec: "If you avoid these 3 things, you'll still be training 3 months from now — most people aren't" - 55-60 sec: CTA — follow for daily gym tips Tone: Warm, encouraging, experienced. Not aggressive. Like the trainer has seen this a hundred times and genuinely wants to help. Write the voiceover in conversational English (can include some Hindi/Marathi words naturally).

90-Second Reel Script Prompt — Story Format

90-sec reels ला narrative structure जास्त काम करतो. Client ची actual transformation story, किंवा trainer चा personal experience. Story format viewers ला hook करतो, longer retain करतो:

90-Sec Story Format Reel PromptWrite a 90-second Instagram Reel script for a gym trainer in Pune, India. Format: Story-driven voiceover — the trainer tells a real-feeling story about a beginner client (anonymized) who made these mistakes and almost quit, but fixed them and transformed. Structure: - 0-5 sec: Hook — start mid-story. "Rahul came to me 6 months ago. He had tried 3 gyms before and quit each time within 2 weeks..." - 5-20 sec: Setup — what Rahul was doing wrong (the 3 mistakes) - 20-55 sec: The turning point — trainer's advice, what changed, specific moment - 55-80 sec: The result — not dramatic, realistic. "3 months later, Rahul hadn't missed a single Monday..." - 80-88 sec: The lesson — what the viewer can take from this - 88-90 sec: CTA — "If you're a Rahul right now, DM me. First session free." Tone: Storytelling, genuine, a little emotional but not overdramatic. Real-feeling. The trainer sounds like someone who actually cares about clients, not someone selling. On-screen visuals: suggest what footage/B-roll to use at each point (gym shots, before/after body language, trainer coaching, etc.)

Voiceover vs Text-on-Screen — कोणता कधी?

हे client च्या situation वर अवलंबून आहे. दोन्हींसाठी prompt approach वेगळा असतो:

Factor Voiceover Text-on-Screen
Client comfortable on camera? हो तर voiceover नाही तर text-on-screen
Prompt instruction "Write as spoken words — conversational script" "Max 5-6 words per frame — fast cuts"
Tone Warm, personal, relationship-building Punchy, quick, high-energy
Works best for Professionals (doctors, trainers, consultants) Transformations, tips lists, quick demos
Editing effort More (syncing audio + visuals) Less (text overlay easy in CapCut)
Practical Tip: नव्या clients साठी text-on-screen reels सुरू करा. त्यांना camera चा comfort नसतो. 2-3 महिन्यांनंतर, जेव्हा trust built होतो, तेव्हा voiceover reels introduce करा. Better content, smoother process.

Exercise — Gym Reel Script बनवा

Part 1: 30-sec prompt वापरा. Topic ठेवा "3 mistakes beginners make." Text-on-screen format. Generate करा, वाचा.

Part 2: Same conversation मध्ये पुढे जा. 60-sec prompt वापरा — same topic, voiceover format. Compare करा: कोणता format तुम्हाला जास्त engaging वाटतो? कोणता client साठी easier होईल बनवायला?

दोन्ही scripts save करा. Reel structure (Hook → Problem → Solution → CTA) दोन्हींमध्ये कसा differ करतो ते notice करा.
