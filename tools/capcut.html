CapCut — Reels बनवायला शिका

Images ला video मध्ये convert करा. 30-second reel ready करा — free, mobile वर.

CapCut म्हणजे काय?

CapCut हा ByteDance (TikTok चीच company) चा free video editor आहे — mobile (Android/iOS) आणि web दोघांवर available आहे. Instagram Reels बनवण्यासाठी सगळ्यात popular free tool आहे. का? कारण तो simple आहे — drag and drop, templates, music library, text animations — सगळं एकाच app मध्ये. Professional video editing software शिकायची गरज नाही. तुमच्या Canva carousel images ला एक छान 30-second reel मध्ये convert करणं CapCut मध्ये 15-20 minutes चं काम आहे.

Download आणि Setup

  1. Mobile साठी: Play Store (Android) किंवा App Store (iOS) मध्ये "CapCut" search करा → Install करा
  2. Web साठी: capcut.com उघडा → "Use for free" → account बनवा
  3. App उघडल्यावर "Sign up" → TikTok, Google, किंवा phone number वापरा
  4. Home screen दिसेल — "New project" button दिसेल, मोठं असतं
Mobile vs Web: Mobile app जास्त features देतो आणि fast आहे. Web version desktop वापरणाऱ्यांसाठी. या guide मध्ये mobile flow follow करतो — web मध्ये buttons same असतात, locations थोड्या different.

Step-by-Step: Canva Images → 30-Second Reel

Step 1 — New Project

  1. CapCut home screen वर "New project" tap करा
  2. Phone gallery उघडेल
  3. Canva मधून download केलेल्या 5 slides select करा (PNG files)
  4. Order check करा — Slide 1 first, Slide 5 last
  5. "Add" tap करा — editor उघडेल

Step 2 — Duration Set करा (5-8 Seconds प्रति Image)

30-second reel साठी 5 images असतील तर प्रत्येक image 6 seconds (5 × 6 = 30 seconds). हे standard आहे — viewer ला reading साठी enough time मिळतो.

  1. Timeline मध्ये पहिला image clip tap करा
  2. Bottom menu मध्ये "Duration" option येईल (किंवा clip च्या edge drag करा)
  3. 6.0 seconds set करा
  4. सगळ्या 5 clips साठी हेच करा
Duration Guide:
  • Text-heavy slides (reading needed): 7-8 seconds
  • Simple single-point slides: 5-6 seconds
  • CTA / cover slides: 4-5 seconds
  • 30-second reel साठी: 4 × 7 + 1 × 2 = 30 seconds किंवा 5 × 6 = 30 seconds

Step 3 — Text Animation Add करा

  1. Timeline मध्ये desired slide position वर scrub (drag) करा
  2. Bottom menu मध्ये "Text" tap करा
  3. "Add text" → text type करा (hook word किंवा slide number)
  4. Text select असताना "Animation" tap करा
  5. Simple animations निवडा: "Fade in," "Typewriter," "Rise" — हे professional दिसतात
  6. Fancy animations (bounce, shake, zoom crazy) टाळा — distracting असतात
  7. Duration: text animation 0.5-1 second असू द्या
Simple > Fancy: Beginners ला fancy effects tempting वाटतात — but professional content creators simple रहातात. Viewer चा focus content वर असला पाहिजे, effects वर नाही. एक rule: जर animation notice होतो तर तो चुकीचा आहे.

Step 4 — Transitions Add करा

  1. Timeline मध्ये दोन clips च्या मध्ये एक छोटं icon दिसतं ("|" shape)
  2. त्यावर tap करा — Transitions menu उघडेल
  3. "Basic" category मधून निवडा: "Fade," "Dissolve," "Wipe Left" — clean transitions
  4. Duration: 0.3-0.5 seconds — कमी, smooth
  5. "Apply to all" option वापरा — consistent transitions

Step 5 — Background Music Add करा

  1. Timeline खाली "Audio" tap करा → "Music" निवडा
  2. CapCut चा free music library उघडेल — हजारो tracks आहेत
  3. Categories explore करा: "Trending," "Chill," "Corporate," "Motivational"
  4. Client च्या niche नुसार निवडा:
    • Dental/Medical: "Calm," "Soft Piano," "Corporate"
    • Gym/Fitness: "Motivational," "Upbeat," "Energetic"
    • Salon/Beauty: "Chill," "Aesthetic," "Soft Pop"
    • Cafe: "Acoustic," "Lo-fi," "Warm"
  5. Track निवडा → "Use" tap करा
  6. Music clip timeline मध्ये येईल — video च्या length ला trim करा

Step 6 — Volume Adjust करा

Music volume critical आहे. Background music असेल (no voiceover) तर 50-70% ठेवा. Voiceover असेल तर music 20-30% — voiceover clearly ऐकू यायला हवं.

  1. Music clip select करा → "Volume" tap करा
  2. Slider drag करा — 25% वर set करा (voiceover साठी)
  3. Headphones लावून preview ऐका — music overpower होतंय का?
  4. Voiceover आणि music balance झाला पाहिजे

Export — High Quality Settings

  1. Top-right corner मध्ये "Export" बटण tap करा (arrow icon)
  2. Settings दिसतील — adjust करा:
    • Resolution: 1080p — Instagram साठी standard
    • Frame rate: 30fps — smooth playback
    • Quality: High
  3. "Export" tap करा — rendering होईल (1-3 minutes)
  4. Phone Gallery मध्ये MP4 file save होईल
  5. Instagram वर upload करा — Reel format निवडा
Copyright Warning: CapCut library मधले songs Instagram वर use करणं safe आहे — ते royalty-free आहेत. पण बाहेरून कोणताही popular Bollywood song किंवा English hit CapCut मध्ये import करून use करू नका — Instagram post automatically mute होऊ शकतो किंवा content remove होऊ शकतो. CapCut library = safe. Baahubali BGM = risky.

Exercise 1 — Carousel → 30-Second Reel

Canva मध्ये बनवलेले 5 dental carousel slides घ्या (Exercise 1 from Canva page) आणि CapCut मध्ये reel बनवा:

  1. CapCut new project → 5 PNG files import करा
  2. प्रत्येक slide: 6 seconds duration set करा
  3. Slide 1 (cover) वर "fade in" text animation add करा
  4. सगळ्या transitions: "Dissolve," 0.4 seconds
  5. Music: "Corporate" category मधून soft background track निवडा
  6. Volume: 50% (no voiceover साठी)
  7. Export: 1080p, High quality
  8. Phone वर play करा — mobile screen वर कसं दिसतं ते बघा

Exercise 2 — Music, Volume, Export

Exercise 1 चाच reel घ्या — music experiment करा:

  1. Music track बदला — "Chill" category मधून वेगळा track निवडा
  2. Volume: 30% set करा
  3. खालील script ला तुम्ही स्वतः record करा (phone च्या voice recorder मध्ये) आणि CapCut मध्ये import करा: "दात स्वच्छ ठेवणं सोपं आहे — फक्त या ५ गोष्टी लक्षात ठेवा."
  4. Voiceover import केल्यावर music volume 20% वर आणा
  5. Export करा → दोन्ही versions compare करा (with/without voiceover)

Learning: Music + voiceover balance कसा असावा हे तुम्हाला practice ने जमतं. Headphones वापरा — speaker वर balance वेगळा वाटतो.
